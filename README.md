# English RMU Networks

This repository contains two Appendix files for the book chapter titled “The Role of Recurrent Multiword Units in Linguistic Generalization: Examples from English.” This chapter is scheduled for publication in the book “Applied Linguistics, Corpora, and Artificial Intelligence: Advancing Language Research and Education,” edited by Prof. Siaw-Fong Chung and published by Springer Nature. To view these two HTML files correctly, please download the files and open them using a standard web browser.
